Full Ceylon port of the popular puzzle game 2048.

NOTE: The project is currently in progress. The basic game engine is implemented, but there is no UI at the moment.

Motivation

As I've played many variations of this game on mobile platforms, I've become increasingly frustrated by the dismal energy performance of most variants -- Almost all variants I've played on Android eat battery like it was candy and heat up the phone in my hand to the point it is painful...

I suspect the reason for this is that all of these games are built on the same base of HTML + JavaScript, which requires the full power of browser VM between the device and the game implementation. My aim is to re implement this game so that it can be used to build native implementations of the game on any of the chosen target platforms.

Another reason to build my own version of the game is to get a game into the Play store that does not show annoying ads. It is simple enough game without any significant server side requirements, so asking money for this feels kind of unfair.

Ceylon language

So I make this implementation of the game in Ceylon language.

Firstly to make something not completely trivial in this beautiful language.

Secondly, to demonstrate how you can use Ceylon to write an app that has (at least) three distinct UI versions based off the same source files:

  • HTML + JavaScript version to be played on the browser
  • JavaFX on JVM version to be played on the desktop
  • Android on Dalvik version for mobile devices
